# 且慢营销 Logo 资产

按**背景明暗**分目录存放，便于在浅色/深色背景下正确调用。

| 文件夹 | 适用背景 | 调用场景 |
|--------|----------|----------|
| [浅色背景](./浅色背景/) | 白、浅灰、浅蓝等 | 常规 H5、白底落地页、浅色头图 |
| [深色背景](./深色背景/) | 深灰、深蓝、黑等 | 深色首屏、暗色弹窗、深色 Banner |

**规范出处**：且慢营销设计规范 → 3.4 Logo 与品牌标识。

**放入步骤**：将导出的且慢/盈米基金 Logo 切图（PNG 或 SVG）按「浅色背景用蓝版、深色背景用白版」放入对应文件夹，文件名建议见各子目录内 README。
